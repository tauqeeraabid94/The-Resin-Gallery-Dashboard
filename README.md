#The Resin Gallery Dashboard – README.md

📌 Project Title
The Resin Gallery Sales Dashboard | Power BI Project

📖 Short Description
This Power BI dashboard provides a complete analysis of sales performance for The Resin Gallery business.
The dashboard helps track total sales, profits, product categories, payment methods, customer demographics, and sales trends through interactive visualizations.
It is designed to help business owners understand customer behavior, monitor performance, and make data-driven decisions.

🛠️ Tech Stack
Power BI – Dashboard Development & Data Visualization.
Microsoft Excel / CSV – Data Storage & Dataset.
DAX – KPI Calculations & Measures.
Power Query – Data Cleaning & Transformation.

📂 Data Source
The dataset includes:
Resin product sales records
Customer gender data
Payment method details
Product category information
Monthly order trends
Profit and making cost analysis

✨ Features & Highlights

🔹 KPI Cards
Total Sales
Total Making Cost
Total Profits
Total Orders

🔹 Interactive Filters
Users can filter dashboard data by:
Category
Order Status
Payment Status
Gender
Month

🔹 Visual Insights
Payment Method Distribution
Gender-wise Customer Analysis
Category-wise Sales Performance
Monthly Sales Trends
Order Tracking & Business Performance

🔹 Business Insights
Identify high-selling resin products
Analyze customer demographics
Track monthly business growth
Compare payment preferences
Monitor overall profitability
